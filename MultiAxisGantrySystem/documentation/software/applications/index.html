Applications

The operation, configuration, and maintenance of the Multi-Axis Gantry System relly on a few key software tools. This section outlines the essential applications you will need to compile firmware, configure the motor drivers, and interact with the hardware via the web interface.

Ensure you have the following software installed and ready:

Software/Tool Version Purpose
Arduino IDE 2.3.7 Compile & flash firmware
ModbusMaster library ≥ 2.0.1 ModbusMaster by Doc Walker (install via Library Manager)
Leadshine Motion Studio v1.4 Driver parameter configuration
Chrome / Edge browser Latest HMI Web Serial API support
