## Object set

See Appendix B for the full object list and vendor links. The benchmark object set includes
plates, bowls, blocks, towels, a button, a pen cup, oven, pepper, whiteboard eraser, and boxes.

## 3D printing requirements

- Camera mount parts: backplate + snap-hooks for the D455 mount.
- Wrist camera mount for the SO-101 end-effector.
- Use PLA or PETG.
- Typical settings: 15% infill, 0.4 mm nozzle.
- Print the STL files referenced in Appendix A before assembly.
