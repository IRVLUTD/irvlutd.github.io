Environment overview

The benchmark uses a low-cost SO-101 follower arm inside a 32x32 in light box with a top camera (RealSense D455) and a wrist RGB camera. An AprilTag provides camera pose alignment for consistent scene replication across labs.

  • Keep lighting, camera angles, and background sheet consistent.
  • Read each checklist item before moving to the next section.

Assemble camera mounts (top camera)

  1. 3D print two snap-hooks (Part1.stl) and one backplate (Part2.stl) in PLA/PETG, 15% infill, 0.4 mm.
  2. Attach both snap-hooks to the backplate with M3x6 mm screws (assembled part = Part3).
  3. Mount Part3 onto the D455 rear holes using two M4x6 mm screws.
  4. Add rubber grip tape inside each hook to prevent sliding.
Checklist
  • Hooks sit flush against the backplate.
  • Camera is secure and does not wobble.
  • Grip tape is applied inside both hooks.

SO-101 and wrist camera setup

  1. Assemble the SO-101 follower arm per LeRobot docs (do not calibrate yet).
  2. Print and install the wrist camera mount (TheRobotStudio guide).
  3. Secure the wrist mount to the end-effector with an M3x12 mm screw and M3 nut.
  • LeRobot SO-101 documentation
  • Wrist camera mount guide

Light box setup

Assemble the 32x32 in light box before adding the tarp. Orient the box so the open face is -y, the top is +z, and the workspace floor is -z.

  1. Build the PVC frame with 12 pipes and 8 edge connectors.
  2. Attach the diffuser sheet to the +z face with velcro, taut and centered.
  3. Mount LED panels: two on +z face (center ~7.5 in from -y and +y), one on +y face (center ~8 in from +z).
  4. Mount the front-view camera at the +z/+y internal junction; camera midpoint ~7.5 in from -x.
  5. Slide the zipper tarp over the frame, aligning the -y face with the zipper side.
  6. Attach the white background sheet to the +y face and tuck under the -z pipes.
  7. Clamp the overhanging background sheet to the table on both sides.
Checklist
  • LED panels are powered and set to max brightness (~5600K).
  • Diffuser sheet is taut with no sag.
  • Background sheet is flat with no wrinkles in the camera view.

Mount the SO-101 follower arm and place AprilTag

  1. Clamp both sides of the SO-101 base to the table. The base front edge touches the PVC pipe between -z and -y.
  2. Align the base center ~16.5 in from the -x face of the box.
  3. Attach the 12 V power adaptor to the SO-101 arm.
  4. Place the 4 cm AprilTag on the right side of the base.
  5. Align the tag: northwest corner touches the vertical edge; south border aligns with the base bottom.
Checklist
  • Base center is 16.5 in from the -x face.
  • Base front edge is flush against the -z/-y PVC pipe.
  • AprilTag is flat with no wrinkles or lifted corners.

Record the camera indices from the output for calibration.

Camera calibration

Run the AprilTag calibration script and adjust the front-view camera until the pose values match the targets below (within tolerance).

X (m) Y (m) Z (m) R (deg) P (deg) Y (deg)
-0.06 +- 0.01 -0.39 +- 0.01 1.25 +- 0.01 -18.5 +- 1.0 3.0 +- 1.0 2.5 +- 1.0 
python calibration/camera/detect_apriltag.py
  1. If no feed appears, update the device index in the script.
  2. Physically slide or tilt the camera until the pose values match.
  3. Press q to exit once aligned.
  4. Run overlay matching for top and wrist cameras:
python calibration/camera/overlay.py --base-cam <id> --top-cam
python calibration/camera/overlay.py --base-cam <id> --wrist-cam
Checklist
  • All six pose values match the targets within tolerance.
  • Overlay images match closely for top and wrist cameras.

Calibrate the SO-101 arm

  1. Follow LeRobot SO-101 calibration steps and ensure motors are centered.
  2. Find the calibration file created by LeRobot:
~/.cache/huggingface/lerobot/calibration/robots/so_follower
  1. Copy the JSON file to VLAReplica/calibration/robots/so101_follower.
  2. Rename it to so101_follower_arm.json.

Policy evaluation (benchmark.py)

Use the CLI below to evaluate a policy across tasks.

python benchmark.py \
  --policy-type pi0 \
  --policy-path lerobot/pi0_base \
  --policy-from-hub \
  --run-all-tasks \
  --task-subset ID \
  --iterations 5 \
  --eval-follower-calib-dirs calibration/robots/so101_follower \
  --eval-follower-ports /dev/ttyACM1 \
  --eval-follower-ids so101_follower_arm \
  --eval-top-indexes 4 \
  --eval-wrist-indexes 14 \
  --reset-mode fixed \
  --reset-action-file arm_reset.json
  • Match the live video feed to the overlay image before pressing Enter.
  • Each iteration runs for up to 90 seconds, or ends early if the task completes.
